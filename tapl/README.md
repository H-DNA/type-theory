# Types and Programming Languages (TAPL)

Exercises from the **TAPL** textbook.

## About the book

- **Author**: Benjamin C. Pierce
- **Published**: 2002
- **Publisher**: MIT Press
- **ISBN**: 978-0262162098

Types and Programming Languages is the definitive textbook on type systems for programming languages. It provides a comprehensive introduction to type systems and their role in language design, implementation, and reasoning about programs.

## Topics covered

- Untyped systems (arithmetic, lambda calculus)
- Simple types
- Subtyping
- Recursive types
- Polymorphism (System F)
- Higher-order systems
- Type reconstruction and inference

## Language

All exercises are written in **OCaml**.
