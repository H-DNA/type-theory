# Software Foundations

Exercises from the **Software Foundations** book series.

## About the book

- **Authors**: Benjamin C. Pierce, Arthur Azevedo de Amorim, Chris Casinghino, Marco Gaboardi, Michael Greenberg, Cătălin Hriţcu, Vilhelm Sjöberg, Andrew Tolmach, Brent Yorgey, and others
- **First Published**: 2007 (continuously updated)
- **Website**: https://softwarefoundations.cis.upenn.edu/

Software Foundations is a series of electronic textbooks on the mathematical underpinnings of reliable software. The series covers topics including logic, computer-assisted theorem proving, functional programming, operational semantics, and verification of practical programs.

## Volumes

- **LF** — Logical Foundations: Introduction to Coq and functional programming
- **PLF** — Programming Language Foundations: Type systems and operational semantics
- **VFA** — Verified Functional Algorithms: Formal verification of data structures
- **QC** — QuickChick: Property-based testing in Coq

## Language

All exercises are written in **Coq**, an interactive theorem prover.
