# Logical Foundations (LF)

Exercises from **Volume 1: Logical Foundations** of the Software Foundations series.

## About the book

- **Authors**: Benjamin C. Pierce et al.
- **Part of**: Software Foundations series
- **Website**: https://softwarefoundations.cis.upenn.edu/lf-current/toc.html

Logical Foundations is an introduction to the foundations of programming language theory using the Coq proof assistant. It covers functional programming, basic concepts of logic, computer-assisted theorem proving, and Coq.

## Topics covered

- Functional programming in Coq
- Inductive data types and pattern matching
- Polymorphism and higher-order functions
- Propositions and proofs
- Logic in Coq
- Induction principles
- Proof tactics

## Language

All exercises are written in **Coq**.
